The overflow-wrap Property

overflow-wrap: normal (default):

a This div contains a very long word: thisisaveryveryveryveryveryverylongword. The long word will not break and wrap to the next line.

overflow-wrap: break-word:

b This div contains a very long word: thisisaveryveryveryveryveryverylongword. The long word will break and wrap to the next line.

overflow-wrap: anywhere:

c This div contains a very long word: thisisaveryveryveryveryveryverylongword. The long word will break and wrap to the next line.
